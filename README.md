# Install
### Publish config
```php
php artisan vendor:publish --tag="laravel-sitemap-config"
```
and specify your sitemap settings in `config/sitemap.php`


### Link to sitemap
Your sitemap.xml file is under http:://127.0.0.1/sitemap.xml`
